### Assumed inches

```

0.5 = 1/2 inch
1/2 = 1/2 inch

```

### Prime marks

```
1' = 1 foot
4 1/2" = 4 1/2 inches
4 1/2' = 4 1/2 feet
2' 6" = 2 feet 6 inches
2' 3" = 2 feet 3 inches
0.5' = 1/2 foot
0.5" = 1/2 inch
2.5' = 2 1/2 feet
```
